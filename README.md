 🐦 Bird Species Recognition

📌 Project Description
Bird Species Recognition is a web-based application that identifies bird species from uploaded images using Artificial Intelligence. The application allows users to upload a bird image and predicts the species with confidence and a short description.

🚀 Features
- Upload bird images
- Preview uploaded image
- AI-based bird species identification
- Displays species name
- Shows confidence level
- Provides a brief justification

 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Tailwind CSS
- Google Gemini API (AI)

📂 Files
- `code5.html` – Main application source code
- `sanjay final report.pdf` – Project report

▶️ How to Run
1. Download the project.
2. Open `code5.html` in any modern web browser.
3. Upload a bird image.
4. Click **Identify Species**.
5. View the predicted bird species and confidence.
